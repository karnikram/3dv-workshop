# 3dv-workshop
Material for the hands-on sessions of the [3D computer vision workshop](http://cvit.iiit.ac.in/workshops/3dvision/) organized by IIIT-H.

## Requirements
- Python 3.7+
- Jupyter notebook
  - `pip3 install jupyter`
- Numpy
  - `pip3 install numpy`
- OpenCV
  - `pip3 install opencv-python`
- Matplotlib
  - `pip3 install matplotlib`
